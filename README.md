📝 Intro Component with Sign-Up Form

A responsive Intro Component with Sign-Up Form built as part of a Frontend Mentor challenge.
The project focuses on combining UI design with form validation and interactivity.

🚀 Features

- Clean two-column layout (text + form)
- User input form (name, email, password)
- Client-side form validation
- Error messages for invalid inputs
- Responsive design (mobile → desktop)
- Modern UI with clear visual hierarchy

| Technology             | Purpose                         |
| ---------------------- | ------------------------------- |
| **HTML5**              | Semantic page structure         |
| **CSS3**               | Styling and layout              |
| **JavaScript**         | Form validation & interactivity |
| **Flexbox / CSS Grid** | Layout alignment                |
| **GitHub Pages**       | Deployment                      |
